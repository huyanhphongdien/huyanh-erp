  • Card "Tự động tính toán" (xanh đen) — y hệt layout hiện có ở Step 1 ERP, đặt ngay trên form để Sale luôn thấy tổng bành / container / giá trị / hoa hồng realtime.
  • Multi-item: 1 SO có thể có nhiều grade (vd SVR10/SBR1502 mixture) — nút "+ Thêm SP" thêm item row, mỗi item tự tính bành/cont/$ riêng, tổng cộng dồn vào card xanh đen + bảng SC/PI.
  • 4 preset thực (Phil Rubber, Apollo, Yoong Do, VITRY) — bấm để load đúng data từ 4 file mẫu .docx đã đọc, form + preview SC/PI cập nhật ngay. Riêng VITRY có Claims 30 ngày + LCIA London → auto-mở khung "Ràng buộc bổ sung".
  • 🔲 So sánh 4 HĐ song song — grid 2×2 + bảng so sánh 16 tiêu chí — dùng khi cần check khác biệt CIF/FOB hoặc giữa các KH (Sale training, BGĐ review). Mỗi card có riêng tab SC/PI.
  • "Ràng buộc bổ sung" = collapse ẩn mặc định. Mở khi cần override Documents/Claims/Arbitration/Freight cho từng KH đặc biệt.
