✅ Tóm tắt thiết kế

📁 Thư mục Tài liệu (6 folder dùng chung)
  • 📋 Hợp đồng — Sale upload, BGĐ + Phú duyệt
  • 🚢 Vận chuyển — Logistics upload (B/L, booking)
  • 📑 Chứng từ — Logistics + KT upload (COA, Phyto, Fumigation, Insurance)
  • 💰 Tài chính — KT upload (LC, Commercial Invoice, Packing List)
  • 🚛 Phiếu cân — auto link từ WMS (readonly)
  • 📦 Khác — admin upload (general)
UX: Widget "📁 Tài liệu" ở tab Tiến độ (default) → thấy ngay tổng quan; mỗi tab có FileSection riêng cho nhóm đó; badge số file trên tab bar.
💬 Chat trao đổi (Khu vực cho mọi người)
  • Tab "Trao đổi" ngang hàng với các tab khác — badge tin mới
  • @mention (gõ @ ra dropdown) — thông báo người được mention
  • Reply threading — phản hồi tin cụ thể
  • Attach file — drag/drop hoặc click 📎
  • Reactions 👍 🎉 ❤️
  • System messages — auto khi có sự kiện (trình, duyệt, ký, promote)
  • Pin — ghim tin quan trọng (bank info, deadline)
  • Read receipts — thấy ai đã đọc
  • Sidebar phải: 📌 ghim · 👥 participants · 📎 files đính kèm chat
Stack tech: Tham khảo project_comments + Supabase realtime channel cho live update.
💡 Quyết định thiết kế:
  • Files dùng doc_type enum trong sales_order_documents → 1 bảng quản lý tất cả file của đơn
  • Chat dùng bảng mới sales_order_messages (mô phỏng project_comments) với realtime subscribe
  • System message auto generated bởi DB trigger khi status thay đổi
  • Tab "Trao đổi" có realtime badge — number tin chưa đọc highlight đỏ
  • Floating bubble (alt) chỉ enable cho mobile / nhỏ màn — desktop dùng tab
